Methods

login(user, username, validPassword, token) → {any}

Авторизация. Выполяняет поиск пользователя по введенному имени пользователя, если пользователь найден, то проверяет валидность введенного пароля и пароля из БД для данного пользователя, в случае успеха пользователь получает токен, с помощью которого осуществляется доступ ко всем API работы с новостями.
Parameters:
Name Type Description
user string пользователь
username string имя пользователя
validPassword string переменная для проверки пароля
token string токен

registration(errors, username, password, candidate, hashPassword) → {any}

API-Регистрация. Выполняет проверку на уникальное имя пользователя, если пользователь не существует, то хэширует пароль при помощи bcrypt и сохраняет данные пользователя в базу данных
Parameters:
Name Type Description
errors any ошибки при валидации
username string имя пользователя
password string пароль
candidate string переменная для проверки существования пользователя с введенным username
hashPassword string хэшированный пароль
